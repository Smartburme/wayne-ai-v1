## Features ✨
- **မြန်မာဘာသာ အထောက်အကူ** - Myanmar language support
- **Multi-Page Interface** - Chat, Settings, and About pages
- **Knowledge Integration** - Connects with Markdown knowledge bases
- **Responsive Design** - Works on mobile and desktop
- **Modern UI** - Clean, professional interface

## Installation 🛠️
```bash
# Clone the repository
git clone https://github.com/Smartburme/wayne-ai-v1.git

# Navigate to project directory
cd wayne-ai-v1

# Install dependencies
npm install

# Start development server
npm run dev
```

## Usage 🚀
1. **Chat Interface**:
   - Ask questions in Myanmar or English
   - Upload images for analysis
   - View conversation history

2. **Settings**:
   - Toggle dark/light mode
   - Change language preferences
   - Manage privacy settings

3. **Knowledge System**:
   - Edit Markdown files in `docs/knowledge/` to update AI knowledge

# Wayne AI - မြန်မာ့ဉာဏ်ရည်တု အကူအညီစနစ်

## စွမ်းရည်များ
- **မြန်မာဘာသာ အထောက်အကူ** - မြန်မာစာဖြင့် မေးမြန်းနိုင်ခြင်း
- **အသုံးပြုရလွယ်ကူသော အင်တာဖေ့စ်** - စကားပြော၊ ဆက်တင်၊ အကြောင်း စာမျက်နှာများ
- **အသိပညာများနှင့် ချိတ်ဆက်မှု** - Markdown ဖိုင်များဖြင့် အသိပညာများထည့်သွင်းနိုင်ခြင်း
- **စမတ်ဖုန်းနှင့် ကွန်ပျူတာတွင် အဆင်ပြေစွာအသုံးပြုနိုင်ခြင်း**

## ထည့်သွင်းနည်း
```bash
# Repository ကို ကလုန်းလုပ်ပါ
git clone https://github.com/Smartburme/wayne-ai-v1.git

# Project ဖိုလ်ဒါထဲသို့ ဝင်ပါ
cd wayne-ai-v1

# လိုအပ်သော ပရိုဂရမ်များကို ထည့်သွင်းပါ
npm install

# ဖွံ့ဖြိုးမှု ဆာဗာကို စတင်ပါ
npm run dev
```

## အသုံးပြုနည်း
1. **စကားပြောစနစ်**:
   - မြန်မာသို့မဟုတ် အင်္ဂလိပ်ဘာသာဖြင့် မေးမြန်းနိုင်သည်
   - ဓာတ်ပုံများတင်ပြီး ဆန်းစစ်နိုင်သည်
   - ယခင်စကားပြောခွက်များကို ပြန်လည်ကြည့်ရှုနိုင်သည်

2. **ဆက်တင်များ**:
   - အမှောင်နှင့် အလင်းများကို ပြောင်းလဲနိုင်ခြင်း
   - ဘာသာစကားရွေးချယ်မှု
   - ကိုယ်ရေးကိုယ်တာ ချိန်ညှိမှုများ

3. **အသိပညာစနစ်**:
   - `docs/knowledge/` ဖိုလ်ဒါအတွင်းရှိ Markdown ဖိုင်များကို တည်းဖြတ်ပြီး AI ၏ အသိပညာများကို အဆင့်မြှင့်နိုင်သည်
